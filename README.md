# Salud Familiar Panel

MVP web responsive para gestionar familiares, medicamentos, horarios, citas y fórmulas.

## Despliegue
Este proyecto está pensado para desplegarse como sitio estático en Vercel.

## Archivos
- `index.html`: interfaz principal del panel.

## Siguientes pasos
1. Conectar `family_members` de Supabase.
2. Insertar medicamentos en `medications`.
3. Insertar horarios en `medication_schedules`.
4. Conectar citas y fórmulas.
